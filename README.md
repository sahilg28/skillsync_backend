# SkillSync Backend

A Node.js backend service for the SkillSync job matching platform. Uses AI to connect job seekers with their ideal opportunities.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start development server
npm run dev
```

## Tech Stack
- Node.js + Express
- MongoDB
- JWT Authentication
- OpenAI API
